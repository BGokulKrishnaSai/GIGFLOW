# GigFlow - Mini Freelance Marketplace Platform

A full-stack MERN (MongoDB, Express, React, Node.js) application built for ServiceHive assignment. GigFlow is a mini freelance marketplace where clients can post jobs (gigs) and freelancers can bid on them.

## ðŸŽ¯ Project Overview

GigFlow allows users to:
- **Post Gigs**: Clients can create job listings with title, description, budget, and required skills
- **Browse Gigs**: Freelancers can search and filter available gigs
- **Submit Bids**: Freelancers can bid on gigs with a message and proposed amount
- **Hire Freelancer**: Clients can review bids and hire the best candidate
- **Track Bids**: Freelancers can monitor their bid status
